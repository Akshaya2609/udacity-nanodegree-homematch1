Udacity-nanodegree-Real estate agent project

The respository consist of the project provided by udacity as a part of udacity-nanodegree. This project
is about creating a real estate agent that generates structured real estate property listings


Dependencies:

pip install openai==0.28 chromadb numpy pandas

Usage:
1. Clone the repository using the following command
   https://github.com/your-username/HomeMatch.github

2.Install the dependencies mentioned in requirements

3.Install colab from extensions and select the python3 kernel

4.Run the code cell by cell to retrieve the output or use runall to get all the output


Project structure:
1.HomeMatch.ipynb-It consist of the code to create a real estate agent
2.listings.txt-It consist of the generated listings
3.requirements.txt-consists of the dependencies to be installed in order to run the project
